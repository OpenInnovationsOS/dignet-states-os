# LexOS — Law Engine, Voting, Courts, AI Governance Assistant
# Governance

This Digital State OS is governed by its citizens.

## How to Participate
- ✅ **File an Issue** to propose a law, fix, or feature  
- ✅ **Submit a Pull Request** to improve code/docs  
- ✅ **Join our Discord** to discuss (link coming soon)

## Decision Making
- Small changes: Maintainer approval  
- Major changes (e.g., new service): Citizen vote via `dnet vote` (coming Q1 2026)
